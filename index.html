IMS - Inventory Management System

This web application is responsible for CRUD(Create Read Update Delete) of Inventory. There are two roles available for user - 1) Admin and 2) Seller. Admin can perform CRUD operations and Data Analysis on inventory and Seller can sell(Update) items from inventory.
